# Vink's Image Converter
[English](README.md) 

Vink's Image Converter è un convertitore di immagini gratuito, attualmente supporta al 100% questi formati
* .webp
* .avif
* .png
* .jpg
* .ico


> [!NOTE]
> Sto ancora lavorando a questo progetto, al momento presenta alcuni problemi importanti con altri formati, come:
> * Quando si tenta di convertire qualsiasi formato con risoluzione > 16384x16384 in .webp il programma restituisce un .zip vuoto, ho già una soluzione pronta per questo problema
> * I formati di stampa come .pdf | .tiff | .bmp sono instabili e sto cercando di sistemarli
> 
> Miglioramenti grafici arriveranno nelle versioni future.
>
> Il motivo alla base di questo progetto è che volevo testare le mie capacità e creare qualcosa che userei davvero nelle mie attività quotidiane.
